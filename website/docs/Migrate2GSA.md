## Overview

Many organizations are already using some kind of solution to provide access to on-prem resources, block web content, etc. Migrate2GSA is a PowerShell-based migration toolkit designed to help organizations transition from these other solutions to Global Secure Access (GSA). This toolkit provides automated configuration export, transformation, and a set of common GSA provisioning tools to streamline the transition process.

Solutions you can migrate from using this tool include:
 - Cisco Umbrella
 - Forcepoint
 - Netskope
 - Citrix NetScaler
 - Zscaler
 - Defender for Endpoint

**For detailed documentation and access to the tool, navigate to https://aka.ms/Migrate2GSA.**
